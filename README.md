# PixelStitch - Crochet Utility App 🧶

## Overview
PixelStitch is a mobile application designed specifically for crocheters to track complex patterns, manage stitch counts, and organize their project inventory. I built this app to solve a common problem within the crafting community, demonstrating my ability to identify user needs and develop practical digital solutions.

## Key Features
*   **[Feature 1, e.g., Interactive Stitch Counter]:** Allows users to tap to increment rows/stitches easily while working.
*   **[Feature 2, e.g., Pattern PDF Viewer]:** Users can upload and view their patterns directly within the app without switching screens.
*   **[Feature 3, e.g., Project Management]:** Keep track of yarn colors, hook sizes, and project progress in one place.

## Technologies Used
*   **Environment:** Android Studio
*   **Language:** Kotlin and CSS
*   **UI/Design:** XML layouts
